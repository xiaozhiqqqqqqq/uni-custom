## 已实现功能

### ✅ 按钮组件 (Button)
- 支持多种类型：`primary`、`secondary`、`success`、`warning`、`danger`
- 支持多种尺寸：`small`、`medium`、`large`
- 支持状态：`disabled`、`plain`、`round`
- 完整的 TypeScript 类型支持
- 响应式设计，支持 hover 和 active 状态

### ✅ 提示点组件 (Dot)
- 支持空状态显示红点
- 支持数字显示，超过99显示"99+"
- 支持自定义缩放比例
- 响应式布局，自适应内容宽度
- 平滑过渡动画效果

### ✅ 自定义复选框组件 (CustomCheck)
- 自定义图标样式设计
- 支持双向数据绑定 (v-model)
- 支持自定义尺寸
- 支持选中状态切换事件
- 完整的 TypeScript 类型支持

### ✅ 自定义头部组件 (CustomHeader)
- 自动适配小程序导航栏
- 内置返回功能集成
- 支持自定义标题
- 支持插槽扩展右侧内容
- 响应式布局适配不同设备

### ✅ 自定义开关组件 (CustomSwitch)
- 支持 boolean 和 number 两种数据类型
- 支持自定义尺寸和颜色主题
- 支持禁用状态和失败回调
- 支持异步状态变更验证
- 双向数据绑定 (v-model) 支持
- 完整的 TypeScript 类型支持

### ✅ 自定义提示框组件 (CustomToast)
- 多种提示类型支持
- 自定义显示位置
- 支持自动消失和持续时间控制
- 全局单例模式

### ✅ 自定义提示气泡组件 (CustomTip)
- 支持上下两种位置
- 支持自定义样式和位置
- 显示/隐藏状态控制
- 适合引导和操作提示场景

### ✅ 构建配置
- 基于 Vite 的现代构建工具链
- 支持 ES 模块和 UMD 模块输出
- 自动生成样式文件
- TypeScript 支持

### ✅ 包管理
- NPM 包配置完整
- 正确的导出字段配置
- 支持 Tree Shaking

## 使用方式

### 1. 开发模式
```bash
npm run dev          # 启动开发服务器
```

### 2. 构建组件库
```bash
npm run build:lib    # 构建为可分发的组件库
```

### 3. 打包分发
```bash
npm pack            # 生成 .tgz 文件
```

### 4. 在其他项目中使用

#### 方式 1: 完整引入
```javascript
import { createApp } from 'vue'
import UniappUI from '@your-org/uniapp-ui'
import '@your-org/uniapp-ui/style.css'

app.use(UniappUI)
```

#### 方式 2: 按需引入
```javascript
import { Button, Dot, CustomCheck, CustomHeader, CustomSwitch } from '@your-org/uniapp-ui'
```

#### 方式 3: CDN 引入
```html
<script src="https://unpkg.com/@your-org/uniapp-ui/dist/index.umd.js"></script>
<link rel="stylesheet" href="https://unpkg.com/@your-org/uniapp-ui/dist/style.css">
```

## 组件特性

### 组件属性表

#### Button 组件
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| type | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger'` | `'primary'` | 按钮类型 |
| size | `'small' \| 'medium' \| 'large'` | `'medium'` | 按钮尺寸 |
| disabled | `boolean` | `false` | 是否禁用 |
| round | `boolean` | `false` | 是否圆角 |
| plain | `boolean` | `false` | 是否朴素按钮 |

#### Dot 组件
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| count | `number \| string` | `''` | 提示数字，空则显示红点 |
| scale | `number` | `1` | 缩放比例 |

#### CustomCheck 组件
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| checked | `boolean` | `false` | 是否选中 |
| size | `number` | `20` | 图标尺寸(px) |

#### CustomHeader 组件
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | `string` | - | 标题文字 |

#### CustomSwitch 组件
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| open | `boolean \| number` | `false` | 开关状态 |
| size | `number` | `20` | 开关尺寸(px) |
| color | `string` | `'#006BFF'` | 开启时的颜色 |
| disabled | `boolean` | `false` | 是否禁用 |
| beforeChange | `() => Promise<boolean>` | - | 状态改变前的异步验证 |

### 事件表
| 组件 | 事件名 | 参数 | 说明 |
|------|-------|------|------|
| Button | click | `(event: MouseEvent)` | 点击事件 |
| CustomCheck | update:checked | `boolean` | 选中状态更新 |
| CustomCheck | change | `boolean` | 选中状态改变 |
| CustomSwitch | update:open | `boolean \| number` | 开关状态更新 |
| CustomSwitch | change | `boolean \| number` | 开关状态改变 |
| CustomSwitch | failed | `boolean \| number` | 状态改变失败 |

## 当前组件概览

目前组件库包含以下 7 个组件：

1. **Button** - 基础按钮组件
2. **Dot** - 消息提示点组件  
3. **CustomCheck** - 自定义复选框组件
4. **CustomHeader** - 自定义头部导航组件
5. **CustomSwitch** - 自定义开关组件
6. **CustomToast** - 自定义提示框组件
7. **CustomTip** - 自定义提示气泡组件
